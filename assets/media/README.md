# Media Files Folder

Place your video and audio files here.

## Folder Structure

```
assets/media/
  video1.mp4          ← Video files
  video2.mp4
  audio1.mp3          ← Audio files
  audio2.mp3
  video1-poster.jpg   ← Optional poster images for videos
```

## File Naming

- Match your `pid` values from your CSV
- Use lowercase and underscores (no spaces)
- Examples: `video1.mp4`, `interview_1.mp3`, `artwork_video.mp4`

## Supported Formats

**Videos:**
- `.mp4` (recommended)
- `.webm`
- `.ogg`

**Audio:**
- `.mp3` (recommended)
- `.ogg`
- `.wav`

## Usage in CSV

Reference files like this in your Google Sheet:
- `/assets/media/video1.mp4`
- `/assets/media/audio1.mp3`
